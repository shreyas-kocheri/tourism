# 🧘 Module 4: Yoga AI Posture Correction

## Overview
This module uses computer vision and machine learning to analyze yoga postures and provide correction feedback.

It helps users improve posture accuracy during yoga practice.

## Key Features
- Real-time posture detection
- Pose comparison with reference models
- Feedback on alignment errors

## Technology Stack
- Python
- OpenCV
- MediaPipe
- Machine Learning Model

## How It Works
1. Camera captures user posture
2. Keypoints are detected
3. Model compares posture
4. Feedback is displayed

## How to Run
```bash
python detect_pose.py
