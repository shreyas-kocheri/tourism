# 🌍 Module 3: Community Intelligence System

## Overview
This module allows local contributors to post real-time tourism updates such as road blocks, weather issues, and crowd conditions.

Users can like, dislike, and comment to validate information.

## Key Features
- Community posts and alerts
- Like and dislike system
- Image uploads for proof
- Public feed view

## Technology Stack
- Python
- FastAPI
- SQLite
- HTML, CSS, JavaScript

## How It Works
1. Local user submits an update
2. Post appears in public feed
3. Community validates via reactions
4. Highly validated posts gain trust

## How to Run
```bash
pip install -r requirements.txt
uvicorn app:app --reload
